# Ejercicio 11 - Hashing Extensible

**Enunciado:** Capacidad 2 registros por bloque.
Claves:

1. + Aconcagua: 10100111 -> 1 (1 bit), 11 (2 bits), 111 (3 bits)
2. + Kilimanjaro: 10101010 -> 0 (1 bit), 10 (2 bits), 010 (3 bits)
3. + Mont Blanc: 00111110 -> 0 (1 bit), 10 (2 bits), 110 (3 bits)
4. + Cervino: 01101111 -> 1 (1 bit), 11 (2 bits), 111 (3 bits)
5. + Etna: 00110101 -> 1 (1 bit), 01 (2 bits), 101 (3 bits)
6. + Chañi: 11110000 -> 0 (1 bit), 00 (2 bits), 000 (3 bits)
7. + Cho Oyu: 01011101 -> 1 (1 bit), 01 (2 bits), 101 (3 bits)
8. + Vinicunca: 01011011 -> 1 (1 bit), 11 (2 bits), 011 (3 bits)
9. + Chañi
10. + Cervino

## 1. Insertar Aconcagua y Kilimanjaro
+ `p = 0`. B0 (`p' = 0`) recibe ambas.

| # Bloque | p' | R1 | R2 |
|---|---|---|---|
| **0** | 0 | Aconcagua | Kilimanjaro |

## 2. Insertar Mont Blanc
+ OVERFLOW en B0. `p = 1`.
+ Último bit: Kilimanjaro (0) y Mont Blanc (0) -> B0 (`p' = 1`). Aconcagua (1) -> B1 (`p' = 1`).

| Directorio (p=1) | # Bloque |
| --- | --- |
| 0 | 0 |
| 1 | 1 |

| # Bloque | p' | R1 | R2 |
| --- | --- | --- | --- |
| **0** | 1 | Kilimanjaro | Mont Blanc |
| **1** | 1 | Aconcagua | |

## 3. Insertar Cervino
+ Sufijo 1 -> B1. Tiene lugar.

| # Bloque | p' | R1 | R2 |
|---|---|---|---|
| **1** | 1 | Aconcagua | Cervino |

## 4. Insertar Etna
+ Sufijo 1 -> B1. OVERFLOW en B1 (`[Aconcagua, Cervino]`).
+ `p` sube a 2.
+ Últimos 2 bits: Etna (01) -> B2 (`p' = 2`). Aconcagua (11) y Cervino (11) -> B1 (`p' = 2`).
+ Aconcagua y Cervino terminan en 11, por lo que quedan juntos en B1. B1 queda lleno con 2 registros, pero el insertado fue Etna (que fue a B2), por lo que el overflow se resuelve y ningún bloque rebasa su capacidad.

| Directorio (p=2) | # Bloque |
| --- | --- |
| 00 | 0 |
| 01 | 2 |
| 10 | 0 |
| 11 | 1 |

| # Bloque | p' | R1 | R2 |
| --- | --- | --- | --- |
| **0** | 1 | Kilimanjaro | Mont Blanc |
| **1** | 2 | Aconcagua | Cervino |
| **2** | 2 | Etna | |

## 5. Insertar Chañi
+ Sufijo 00 -> B0. OVERFLOW en B0 (`[Kilimanjaro, Mont Blanc]`).
+ `p' = 1` y `p = 2`. Dividimos B0, su `p'` sube a 2.
+ Últimos 2 bits: Chañi (00) -> B0 (`p' = 2`). Kilimanjaro (10) y Mont Blanc (10) -> B3 (`p' = 2`).

| Directorio (p=2) | # Bloque |
| --- | --- |
| 00 | 0 |
| 01 | 2 |
| 10 | 3 |
| 11 | 1 |

| # Bloque | p' | R1 | R2 |
| --- | --- | --- | --- |
| **0** | 2 | Chañi | |
| **1** | 2 | Aconcagua | Cervino |
| **2** | 2 | Etna | |
| **3** | 2 | Kilimanjaro | Mont Blanc |

## 6. Insertar Cho Oyu
+ Sufijo 01 -> B2. Tiene lugar.

| # Bloque | p' | R1 | R2 |
|---|---|---|---|
| **2** | 2 | Etna | Cho Oyu |

## 7. Insertar Vinicunca
+ Sufijo 11 -> B1 (`[Aconcagua, Cervino]`). OVERFLOW en B1.
+ `p'=2` y `p=2`. `p` sube a 3.
+ Últimos 3 bits: Vinicunca (011) -> B4 (`p' = 3`). Aconcagua (111) y Cervino (111) -> B1 (`p' = 3`).

| Directorio (p=3) | # Bloque |
| --- | --- |
| 000 | 0 |
| 001 | 2 |
| 010 | 3 |
| 011 | 4 |
| 100 | 0 |
| 101 | 2 |
| 110 | 3 |
| 111 | 1 |

| # Bloque | p' | R1 | R2 |
| --- | --- | --- | --- |
| **0** | 2 | Chañi | |
| **1** | 3 | Aconcagua | Cervino |
| **2** | 2 | Etna | Cho Oyu |
| **3** | 2 | Kilimanjaro | Mont Blanc |
| **4** | 3 | Vinicunca | |

## 8. Eliminar Chañi y Cervino
+ Chañi (000) se elimina de B0. Queda vacío.
+ Cervino (111) se elimina de B1. Queda con Aconcagua.
